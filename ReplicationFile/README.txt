Grimmer and Stewart 
``Text as Data: The Promise and Pitfalls of Automatic Content Analysis Methods for Political Texts"
7/9/2012

This folder contains replication materials for the Text as Data article by Grimmer and Stewart in Political Analysis. All analyses were performed in the R Statistical Computing Language Version 2.15.0

There are three folders each with code and data for replicating portions of the paper: 
ClusterValidationExamples- replicates Figures 3-6 on validations of unsupervised clustering
ScalingExample- replicates Figure 7 of scaling using Wordfish
SupervisedExample- replicates the Supervised Learning example using the Random Forest classifier.

Any additional questions can be sent to Justin Grimmer or Brandon Stewart.  Current contact info can be found by searching for our respective websites.
